# Car Racing Game 🏎️

A 3D car racing game built with React and Three.js!

## Features

- **3D Graphics**: Built with Three.js for smooth 3D rendering
- **Real-time Controls**: Drive the car with keyboard controls
- **Dynamic Camera**: Camera follows the car as you drive
- **Track Environment**: Race on a circular track with realistic lighting
- **Physics**: Basic car physics with acceleration, braking, and turning

## Controls

- **W / ↑**: Accelerate forward
- **S / ↓**: Brake / Reverse
- **A / ←**: Turn left
- **D / →**: Turn right

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5174`

### Build for Production

```bash
npm run build
npm run preview
```

## Technologies Used

- **React**: UI framework
- **Three.js**: 3D graphics library
- **Vite**: Build tool and development server
- **JavaScript ES6+**: Modern JavaScript features

## Game Mechanics

- The car accelerates and decelerates realistically
- Turning is more effective at higher speeds
- The camera follows the car smoothly
- The track is a continuous oval loop
